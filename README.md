# Config Your Vim

## usage

#### Install

**vim**

ubuntu `sudo apt install vim`

arch `sudo pacman -S vim`

centos `yum install vim -y`

**vim-plug**
```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```
---

## 插件

- **gruvbox** - 主题
- **vim-floaterm** - 浮动终端窗口
- **indentLine** - 缩进提示线
